Extension Submission Details
============================

Name: ZeroConnectWallet
Summary: Read-only wallet extension for connecting to DeFi apps without storing private keys
Description:
  ZeroConnectWallet mimics MetaMask's EIP-1193 provider API for read-only wallet 
  connections. Connect to DeFi applications to view balances and portfolio data 
  without storing private keys.
  
  Features:
  - Read-only wallet (no private keys stored, ever)
  - MetaMask-compatible provider injection
  - Multiple address support
  - Firefox Sync support
  - Chain support: Ethereum, Polygon, Arbitrum, Optimism, Base, Sepolia
  - Automatic rejection of signing requests with user notification
  
  This extension is designed for users who want to:
  - View DeFi portfolio data without importing private keys
  - Monitor multiple wallet addresses
  - Connect to dApps in read-only mode
