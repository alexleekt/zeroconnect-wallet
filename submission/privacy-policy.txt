ZeroConnectWallet Privacy Policy
================================

Last Updated: March 24, 2026

Data Collection
---------------
This extension does NOT collect, transmit, or share any user data.

All data is stored locally in your browser:
- Wallet addresses you configure
- Selected network/chain ID
- Custom RPC endpoint (if provided)
- Last connected address

No data is sent to any servers except:
- Read-only RPC calls to your configured Ethereum endpoint (default: https://eth.llamarpc.com)

Data Storage
------------
All data is stored using Firefox's built-in storage APIs:
- browser.storage.sync (Firefox Sync, if enabled)
- browser.storage.local (fallback)

Your data never leaves your device except via Firefox Sync (if you have it enabled).

No Private Keys
---------------
This extension is read-only by design. It does NOT:
- Store private keys
- Store seed phrases
- Store passwords
- Sign transactions
- Sign messages

Third Parties
-------------
This extension does not use any third-party services, analytics, or tracking.

Changes to This Policy
----------------------
If we make changes to this privacy policy, we will update this document and the extension version.

Contact
-------
For privacy questions, open an issue on GitHub: https://github.com/alexleekt/zeroconnect-wallet/issues
